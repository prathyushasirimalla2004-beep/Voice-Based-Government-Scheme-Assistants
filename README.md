# 🏛️ AP & TS Scheme Assistant V2 — 34 పథకాలు

> Telugu Voice-enabled Chatbot for Andhra Pradesh & Telangana Government Schemes

---

## ✨ Features

- ✅ **34 Real Government Schemes** — AP: 23 + TS: 11
- 📂 **123 Dataset Documents** — AP: 82 docs + TS: 41 docs
- 🎙️ **Voice Input** (Chrome only, Telugu `te-IN`)
- 🔊 **Voice Output** (gTTS Telugu TTS)
- 📋 **Category-wise Scheme Listing**
- 🔍 **Smart Keyword Search** — State-context aware (AP vs TS)
- 🌙 **Dark Premium UI**
- 💯 **100% Accuracy** — 30/30 test cases passed

---

## 🚀 Run

```bash
pip install flask gtts
python app.py
# Open: http://localhost:5000
```

## 🗂️ Project Structure

```
SchemeBotV2/
├── app.py                  ← Flask backend (34 schemes, dataset-sourced content)
├── requirements.txt
├── templates/
│   └── index.html          ← Dark theme UI (Telugu, category filter, voice)
├── voice/
│   ├── __init__.py
│   └── tts.py              ← gTTS Telugu/Hindi/English TTS
└── dataset/
    ├── andhra-pradesh/     ← 82 AP government documents
    └── telangana/          ← 41 TS government documents
```

## 🛠️ Tech Stack

| Component | Technology |
|---|---|
| Backend | Python Flask |
| NLP / Search | Rule-Based Keyword Scoring (no ML, no paid API) |
| Voice Input (STT) | Web Speech API — Chrome, `te-IN` |
| Voice Output (TTS) | gTTS — Google Text-to-Speech (`lang='te'`) |
| Frontend | HTML + CSS + JavaScript (Vanilla) |
| Dataset | 123 real government documents (AP: 82 + TS: 41) |
