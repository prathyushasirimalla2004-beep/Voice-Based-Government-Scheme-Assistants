# 🏛️ AP Scheme Assistant V2 — 23 పథకాలు

## Features
- ✅ **23 Real AP Government Schemes** from actual dataset
- 📂 **82 Dataset documents** used for search
- ⌨️ Text mode typing
- 🎙️ Voice mode (Chrome only, Telugu)
- 📋 Category-wise scheme listing
- 🎨 Dark premium UI

## Run
```bash
pip install flask
python app.py
# Open: http://localhost:5000
```
